# PGL-MyList-Books

Este proyecto es una práctica para crear una lista de elementos con propiedades como parte de la asignatura **Programación Multimedia y Dispositivos Móviles**.

El objetivo de este proyecto es el manejo de ítems en una lista, incluyendo su creación, visualización y eliminación. En este caso, se ha decidido trabajar con una lista de libros.
